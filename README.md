# charon
Prepare interface to monitor and increase power and set normal managed mode

Installation:
-------------

```
pip3 install dante-charon
```

Usage (In terminal):
------

* get current state:
```
charon -s
```
* set max power, stop NetworkManage, set monitor mode:
```
charon -p
```
* set managed mode, start NetworkManager
```
charon -n
```
